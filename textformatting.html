Now, here we are taking about some more text formatting features in HTML.

Let's go down one by one


1.The HTML small element represents side-comments and small print, like copyright and legal text, independent of its styled presentation. By default, it renders text within it one font-size smaller, such as from small to x-small. Or simply we can say that, this tag is used to display very small text messages on web page.
Example: This is some normal text.
This is some smaller text.

2.The HTML mark tag is used to mark or highlight text that is of special interest or relevance in an HTML document. Browsers traditionally render the text found within the mark tag as text with a yellow background color. This tag is also commonly referred to as the mark element.
Example: Let's focus on Studies today.

3.The sub tag defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font. Subscript text can be used for chemical formulas, like H2O. Tip: Use the sup tag to define superscripted text.
Example:Some chemistry formulae H2S04, H3PO4

4.The HTML Superscript element 'sup' specifies inline text which is to be displayed as superscript for solely typographical reasons. Superscripts are usually rendered with a raised baseline using smaller text.
Example:maths formula X2+Y2+2XY=(X+Y)2
